# Agile flight authority baseline versus candidate

Date: 2026-06-23

This report compares the frozen playable ACRO baseline `legacy_heavy_racing_quad` with the explicit candidate `5inch_agile_candidate`.

Scope:

- `legacy_heavy_racing_quad` remains the source and normal-build default.
- `5inch_agile_candidate` is selectable explicitly with `/fpvdrone debug playablepreset 5inch_agile_candidate` or by installing the candidate test jar.
- ANGLE and HORIZON are intentionally unchanged.
- Golden traces are not regenerated by these metrics.
- RPM display now distinguishes estimated mechanical RPM from `rotorReferenceRpm`, which is only an internal nondimensional aerodynamic reference.

## Summary

| Metric | legacy_heavy_racing_quad | 5inch_agile_candidate |
| --- | ---: | ---: |
| Static hover full-roll duration | 2.000 s | 0.550 s |
| 20 m/s forward full-roll duration | 2.150 s | 0.600 s |
| Static hover max achieved roll rate | 187.048 deg/s | 676.099 deg/s |
| 20 m/s forward max achieved roll rate | 185.717 deg/s | 650.527 deg/s |
| 20 m/s, -8 m/s sink pullout loss | 3.454 m | 1.303 m |
| 20 m/s pullout max effective T/W | 3.407 | 4.401 |
| 20 m/s pullout minimum advance scale | 0.500 | 0.725 |

The first major legacy bottleneck is the combined effective thrust scale, especially advance-ratio thrust loss at forward speed. The candidate raises ACRO rate authority without increasing displayed RPM and reduces repeated thrust-loss stacking for the candidate profile only.

## Build Artifacts

Run:

```powershell
.\gradlew.bat --no-daemon :neoforge-mod:verifyNeoForgePackaging
```

Outputs:

- `neoforge-mod/build/fpvdrone-distributions/legacy-heavy-racing-quad.jar`
- `neoforge-mod/build/fpvdrone-distributions/5inch-agile-candidate.jar`

The normal NeoForge jar and `legacy-heavy-racing-quad.jar` use `legacy_heavy_racing_quad` as the default playable preset. The candidate jar carries a resource default of `5inch_agile_candidate`; a GameMaster can also select that preset at runtime with `/fpvdrone debug playablepreset 5inch_agile_candidate`.
